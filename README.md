
# FASTREP -- Blazing fast O(logn) string repetition
## Installation
```sh
npm install fastrep
```

## Usage
```js
var fastrep = require('fastrep')
console.log(fastrep.fastrep('hi there!', 5); // hi there!hi there!hi there!hi there!hi there!
```
